## PRÁTICA 02
Escreva um algoritmo em linguagem C que atenda os seguintes requisitos:<br><br>
✓Crie um vetor com a quantidade de dígitos do seu RU.<br>
✓Solicite que usuário digite o seu RU, cada digito digitado deve ser armazenado em uma posição do vetor.<br>
✓Utilizando ponteiros verifique o valor da soma de todos os elementos contidos neste vetor e imprima na tela.<br><br>
Para demonstrar o funcionamento faça as capturas de tela do terminal utilizando seu RU.<br>
